Installer la démo :

Afin d'installer la démo, vous devez tout d'abord exécuter les scripts "createBDD.sql" puis "AddData.sql" pour créer la base de données qui sera utilisée pour le site et ajouter des premières données de test. Ensuite, ouvrez le fichier "accueil.php", préalablement placé sur votre serveur PHP, pour arriver sur la page d'accueil.

Remarque :

Selon le serveur web que vous utilisez, un mot de passe pour se connecter à la base de données peut être nécessaire. Si c'est le cas, il faut rajouter ce mot de passe dans tous les fichiers php afin de pouvoir utiliser la base de données. Ces fichiers utilisent déjà le login "root" et le mot de passe "root" par défaut.
